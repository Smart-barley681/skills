## 🚀 What this is

**skills** is a skill pack for AI coding agents. It helps an AI tool follow clear steps when it writes code, edits files, and runs common tasks.

This project works well with tools like Claude Code, OpenClaw, and other coding agents. It gives them a shared set of skills so they can handle work in a more steady way.

Use it when you want your AI helper to:

- follow the same process each time
- work with fewer mistakes
- switch between tasks with less setup
- keep its work easier to review

## 📦 What you get

This repo includes a set of reusable skills and scripts for AI-assisted coding work. The content is meant to help with tasks like:

- project setup
- file edits
- code checks
- automation steps
- task-specific work flows
- repeatable dev actions

It is built for people who use AI coding agents and want a simple skill library they can apply across projects.

## 🧭 How to use it

Once you have the files on your PC, use them as part of your AI coding workflow.

A common flow looks like this:

1. Open the skill files in your project or tool folder.
2. Point your AI coding agent to the skill set.
3. Ask the agent to follow the skill that fits your task.
4. Let it handle the steps in the skill.
5. Review the result before you keep the changes.

You can use skills for small jobs, such as file edits, or for larger jobs, such as setting up a full task plan.
